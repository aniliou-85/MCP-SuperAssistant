# MCP SuperAssistant Project Purpose

MCP SuperAssistant is a Chrome extension that integrates the Model Context Protocol (MCP) tools with AI platforms like ChatGPT, Perplexity, Google Gemini, Google AI Studio, Grok, OpenRouter, DeepSeek, Kagi, and T3 Chat.

## Key Features:
- **Multiple AI Platform Support**: Works with various AI chat platforms
- **Sidebar UI**: Clean, unobtrusive interface that integrates with AI platforms
- **Tool Detection**: Automatically detects MCP tool calls in AI responses
- **Tool Execution**: Execute MCP tools with a single click
- **Tool Result Integration**: Seamlessly insert tool execution results back into AI conversations
- **Auto-Execute Mode**: Automatically execute detected tools
- **Auto-Submit Mode**: Automatically submit chat input after result insertion
- **Push Content Mode**: Option to push page content instead of overlaying
- **Preferences Persistence**: Remembers sidebar position, size, and settings
- **Dark/Light Mode Support**: Adapts to the AI platform's theme

## Target Platforms:
ChatGPT, Google Gemini, Perplexity, Grok, Google AI Studio, OpenRouter Chat, DeepSeek, Kagi, T3 Chat

## Distribution:
Available on Chrome Web Store and can be manually installed for development.
